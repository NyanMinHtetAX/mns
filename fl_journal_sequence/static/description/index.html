Journal Sequence

This module allows an option to add sequence configuration for journal sequence and refund journal sequence.
User can easily configure and change the invoice/bill sequence number from the journals.
Allowed to create and modify journal sequence.
Allowed to create and modify refund journal sequence.
Work as it is in Odoo v13.

Go to Invoicing > Configuration > Accounting > Journals

In journal, under Sequence Settings page user have to create/select sequence and refund sequence

Created/selected sequence and refund sequence in form

Journal Sequence

Customer invoice with sequence number

Refund Journal Sequence

Customer credit note with sequence number

Note: This module is tested and working in Odoo Community Edition but not tested in Enterprise Edition.
