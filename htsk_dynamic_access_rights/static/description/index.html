Overview

Configuring access rights in Odoo is quite technical for someone who has little experience with the system and can get messy if you are not sure what you are doing. This module helps you avoid all this complexity by providing you with a user friendly interface from where yu can define access to specific objects such as buttons and pages/tabs. Also the app helps you add/remove users to and from any group(s) in batch and with much ease

Configuration

Install the application and go to the general settings, open the administrator/user and change the dynamic access rights settings to 'Configuring Dynamic Access'. This option gives the administrator/user administrator rights over the application else leave it blank for other users.

How it works

    Return to the main dashboard and open the app



  • Create a new access configuration and fill all the fields. Note: buttons and pages are discovered automatically(you don't have to create them


    • Before hiding buttons





    After hiding buttons





  • To add/remove users to/from groups, click on the Group Assignment menu, create a new assignment rule and save the record. This will add/remove the selected users to/from the selected groups
